Introduction

EdgeMultiplay is a sample for Creating Multiplayer experiences on Unity using servers deployed on MobiledgeX Cloudlets The solution consists of an EdgeMultiplay server and EdgeMultiplay Unity client /section EdgeMultiplayServer The EdgeMultiplay server does two specific functions:

  • MatchMaking (Creating rooms & assigning Players to rooms).
  • Forwarding messages between players.

This is the documentation of the EdgeMultiplay Unity client
